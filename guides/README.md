# 📚 GUIDES — Guías Operacionales

**Bienvenido a las guías de usuario de CertTrack IT.**

Este directorio contiene manuales paso-a-paso para usuarios finales (Admins y Empleados).

---

## 📖 Guías Disponibles

### 👨‍💼 [GUIA-USUARIO-ADMIN.md](./GUIA-USUARIO-ADMIN.md)

**Para:** Administradores de CertTrack  
**Contiene:**
- Cómo acceder a la app
- Visión general de dashboard
- Gestión de certificaciones (crear, editar, eliminar)
- Gestión de maestros (catálogos)
- Gestión de empleados
- Roadmap futuro
- Reportes & análisis
- Alertas y resolución
- Mejor prácticas

**Tiempo lectura:** 20-30 minutos

---

### 👤 [GUIA-USUARIO-EMPLEADO.md](./GUIA-USUARIO-EMPLEADO.md)

**Para:** Empleados (usuarios finales)  
**Contiene:**
- Cómo acceder a la app
- Dashboard personal (Mis Certificaciones)
- Ver certificaciones vigentes
- Actualizar datos de certificación
- Ver alertas de vencimiento
- Entender estados de certificados
- Skills matrix personal
- Preguntas frecuentes

**Tiempo lectura:** 10-15 minutos

---

### ❓ [FAQ-TROUBLESHOOTING.md](./FAQ-TROUBLESHOOTING.md)

**Para:** Todos  
**Contiene:**
- Preguntas frecuentes
- Errores comunes y soluciones
- Troubleshooting por síntoma
- Cuándo contactar soporte

**Tiempo lectura:** 5-10 minutos

## 🎯 Flujo por Rol

### Administrador (Primeros pasos)

```
1. Leer: GUIA-USUARIO-ADMIN.md (30 min)
2. Practicar: Crear 1 certificación de prueba
3. Referencia: FAQ-TROUBLESHOOTING.md si hay dudas
4. Contactar: Tech lead si algún problema
```

### Empleado (Primeros pasos)

```
1. Leer: GUIA-USUARIO-EMPLEADO.md (15 min)
2. Practicar: Ver mis certificaciones
3. Referencia: FAQ-TROUBLESHOOTING.md si hay dudas
4. Contactar: Tu admin o support si hay dudas
```

## 🤔 Estructura de Cada Guía

Todas las guías siguen este formato:

```
1. Introducción
   ↓
2. Requisitos (qué necesitas antes de empezar)
   ↓
3. Pasos detallados (screenshots si posible)
   ↓
4. Ejemplos / Casos de uso
   ↓
5. Tips & Best Practices
   ↓
6. Errores comunes y cómo evitarlos
   ↓
7. FAQ específica de este tema
```
